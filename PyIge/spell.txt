              Проверка орфографии в PyIge

У вас должен быть установлен редактор (см install.txt) и система проверки 
орфографии с русским словарем (например, aspell). Не проверял, но должны 
работать и др. системы орфографии, если они имеют аналогичный интерфейс.

Запустите редактор (если вы этого еще не делали), чтобы создался файл 
конфигурации %userprofile%\.PyIge\pyige.conf   (~/.PyIge/pyige.conf в Linux),
выйдите из редактора и добавьте в конец файла конфигурации раздел [spell].

Конечно, настройка этого раздела должна быть через диалоговое окно в меню
Настройки\Настройки PyIGE, но с этим я пока не разобрался. Может быть кто-то
поможет? Пока придется править файл конфигурации вручную...

Например, у меня в Windows работает с такими параметрами:

[spell]
path="C:/Program Files/Aspell/bin/aspell -a --lang=ru_RU --encoding=koi8-r"
coding=koi8-r

Если у вас путь к aspell прописан в PATH, то параметр path можно сократить:

[spell]
path="aspell -a --lang=ru_RU --encoding=koi8-r"
coding=koi8-r

Предыдущие параметры были такие (попробуйте поставить их, если у вас проверка 
работала в предыдущих версиях редактора):

[spell]
path=aspell
options=-a
coding=utf-8

------------------------------------------------------------------
                  ИСПОЛЬЗОВАНИЕ
Использование не отличается от того, что было в предыдущих версиях. 
Проверяются только русские слова (в которых, возможно, есть английские буквы 
или цифры).

При запуске редактора, у меня в Windows появляется дополнительное черное окно 
aspell. Закрывать его нельзя, а то не будет работать проверка орфографии. Если 
случайно закрыли, а проверка орфографии нужна, придется перезагружать редактор.

Для проверки орфографии на вкладке "Код"  в верхней строке справа нажмите кнопку
[АВСv]. Если появится сообщение "Проверка орфографии недоступна. Проверьте, что
aspell правильно установлен", проверьте параметры раздела spell в 
конфигурационном файле, как написано выше. При включенной проверке доступны 
также 2 кнопки, расположенные правее:   "Исправить слово" (карандашик) и 
"Добавить в словарь" (стрелка).

Если проверка включена, то на вкладке "Код" очередное ошибочное слово будет 
выделено подсветкой и подчеркиванием красной волнистой чертой.
Наведите на такое слово курсор мыши - появится список возможных замен. 
К сожалению, этот список доступен только для показа, выбрать из него  слово 
мышкой для совершения замены нельзя. 
Сейчас пользователю доступны 3 варианта:
1)	Нажать кнопку [АВСv], чтобы выключить проверку орфографии. Выделение 
ошибочного слова при этом пропадет.
2)	Сделать по ошибочному слову двойной щелчок или нажать кнопку с 
карандашиком "Исправить слово". Тогда вы переключитесь на одну из четырех 
вкладок, где можно исправить ошибочное слово (оно будет выделено). 
После переключения обратно на вкладку "Код" увидите место очередной ошибки.
3)	Нажать кнопку "Добавить в словарь". Тогда вам будет показано очередное 
ошибочное слово, а данное будет добавлено в файл ignore.txt (расположен там же,
где файл параметров pyige.conf ). Слова из этого файла считаются правильными и
будут пропущены при дальнейшей проверке. Файл ignore.txt это обычный текстовый
файл в кодировке utf-8, его можно редактировать в любом текстовом редакторе, 
поддерживающем эту кодировку.

По окончании проверки появится соответствующее сообщение и кнопка [АВСv] 
автоматически отключится. Можно выбрать на вкладке "Код" другой файл и проверить
его.
